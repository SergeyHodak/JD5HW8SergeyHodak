Веб-програма, яка дозволяє:
    Виконавати реєстрацію користувача.
    Виконавати авторизацію користувача.
    Ведення списку виробників і товарів.
    Перегляд списків товарів та виробників.
    Надає можливість використовувати CRUD операції, над слідуючими об'єктами:
        Role,
        User,
        Product,
        Manufacturer.

Інструкція дійсна станом на 2022.09.01.
1. Завантажте репозиторій з гітхабу у вигляді стисненого файлу.
    У гілці мастер, в корені репозиторію, клацніть на Code.
    У випадаючому меню клацніть на Download ZIP.
2. Помістіт завантажений файл у потрібну папку для розгортання проекту.
    Створіть нову папку, наприклад shop.
    Пимістіть завантажений файл у цю папку.
    Розархівуйте всі файли з JD5HW8SergeyHodak-master в папрку shop.
    Після цього ви можете видалити zip-файл.
3. Після запуску Intellij IDEA відкрийте проект із репозиторію shop.
    File/Open.../ виберіть папку shop/ натисніть OK.
    Натисніть Trust Project.
4. Чекаємо, поки Intellij IDEA завантажить з Інтернету все необхідне для роботи з проектом.
5. Запуск програми.
    Відкрийте клас Application, розташований за адресою src/main/java/goit/ Application .java
    Клацніть по симву запуску, навпроти рядка номер 7 або 8 (різниці не має) виберіть run.
6. Зачекайте, поки программа збереться та запуститься.
    Ознакою цього повинні бути два рядки в кінці, один з яких вкаже вам, який локальний хост вам призначено, за замовчуванням 8080.
    Наприклад: "Tomcat started on port(s): 8080 (http) with context path".
7. Відкрийте браузер і перейдіть за посиланням.
    Посилання localhost:8080
8. Увійдіть як адміністратор, щоб налаштувати програму.
    логін: admin
    пароль: admin
9. Role
    тут ви можете (доступно лише тільки для користувача з ролью ADMIN):
        реєструвати нові ролі,
        редагувати існуючі ролі,
        видалити непотрібні ролі.
10. User
    тут ви можете (доступно лише тільки для користувача з ролью ADMIN):
        зареєструвати нового користувача (можна встановити кілька ролей, розділяючи їх комою),
        редагувати існуючих користувачів (розділяти ролі комою),
        видалити користувачів.
11. Product
    тут ви можете (доступно лише тільки для користувача з ролью ADMIN):
        реєстрація нових товарів з можливістю вибору виробника з випадаючого списку зареєстрованих,
        редагувати,
        видалити.
    Для всіх інших ролей лише перегляд записів.
12. Manufacturer
    тут ви можете (доступно лише тільки для користувача з ролью ADMIN):
        реєструвати нових виробників,
        редагувати існуючі,
        видалити існуючі.
    Для всіх інших ролей лише перегляд записів.
13. На сторінці HOMEPAGE є кнопка для зміни профілю.
